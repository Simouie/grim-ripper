PURPOSE

This Python program copies all WAV files from one folder to another folder.
The hierarchy within the specified folder will be copied over as well.

REQUIREMENTS

You need to have Python installed to use this Python program.
Some experience with command line tools would be nice too.

INSTRUCTIONS

Enter something like this in Command Prompt:
python sound_clone.py <full-path-to-input-directory> <full-path-to-output-directory>

full-path-to-input-directory: path to directory that contains WAV files
full-path-to-output-directory: path to directory where copies of WAV files should go
